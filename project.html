新冠肺炎患者病情分析与预测

  • 基于真实的多中心近千例新冠肺炎患者电子病历数据，分析患者病情，预测患者重症率等。
  • 研究内容： - 多中心新冠肺炎数据去隐私化及预处理 - 新冠肺炎患者人口学信息分析 - 新冠肺炎患者重症率预测模型 - 新冠肺炎患者机械通气率、ICU率等预测
  • 合作单位：浙江省人民医院

肾病关键指标分析与预测

  • 基于多中心肾病患者电子病历数据，分析肾病关键致病因素，基于数据驱动，探索肾小球过滤率预测的关键指标及指标预测模型。
  • 主要内容： - 多中心肾病数据处理 - 预测结局 - 预测肾小球过滤率 - 形成预测肾小球过滤率预测的关键模型
  • 合作单位：中山大学第三附属医院

中医痛风发作因素分析与复发预测等

  • 面向中医数据，开展中医知识图谱构建、中医预问诊问答系统研发、中医机器学习型、中医信息抽取方法等研究，在岭南湿症、痛风合并肾功能下降、肾病复发、皮肤病发病因素及复发等方面开展合作研究。
  • 研究内容： - 中医知识图谱构建及应用 - 中医智能问答模型、信息抽取模型及应用 - 痛风发病因素分析及复发预测 - 皮肤病发病因素分析与复发预测
  • 合作单位：广东省中医院

呼吸系统急危重症临床数据分析与挖掘

  • 基于呼吸系统急危重症（慢性阻塞性肺病等）电子病历数据，研究致病因素分析、风险分析，并开展患者病情阶段预测研究等。
  • 主要内容： - 呼吸系统急危重症临床数据统计分析 - 呼吸系统急危重症临床数据挖掘、风险预警模型 - 呼吸系统疾病致病因素分析、因素关联性分析等
  • 合作单位：广州医科大学附属医院

电子病历文本信息抽取与结构化

  • 建立基于自然语言的肺癌医疗文本识别与结构化处理系统，自动识别肺癌临床医疗文本，并按规定格式进行结构化处理得到结构化数据，文本识别与结构化正确率满足临床使用需求等。
  • 主要内容： - 医学命名实体识别 - 医学实体关系抽取 - 医学实体Normalization
  • 合作单位：中山大学附属肿瘤医院

CASTOR临床试验数据分析与挖掘

  • 面向急性缺血性脑卒中，分析用于评价治疗效果的关键评分指标，分析脑卒中药物的有效性和适用人群特点。
  • 研究内容： - NIHSS评分的降分率分析 - mRS评分分析 - EQ-5D量表评分分析 - MMSE量表评分分析 - CASTOR数据挖掘
  • 合作单位：天普药业

区域教育治理大数据分析与可视化平台

  • 构建面向区域教育治理的大数据平台，并基于平台进行教育治理数据的监测、分析、预测以及可视化等。
  • 主要内容： - 指标体系权重动态测算算法 - 面向教育大数据的分析与挖掘 - 教育大数据动态监测与可视化 - 面向教育大数据治理的预测预警模型
  • 合作单位：粤港澳大湾区教育发展高等研究院

基于人工智能的教育领域课堂对话分析与挖掘

  • 面向课堂对话多模态数据，分析课堂对话的类别，挖掘名师课堂对话的模式和规律，评估教师的课堂对话质量及效果。
  • 主要内容： - 面向机器分析的课堂对话质量评估的设计模型 - 课堂对话的数据采集、预处理（语音-文本） - 课堂对话的分析与挖掘（学生的学习情况、教师授课情况、知识掌握情况等） - 课堂对话质量的评估及反馈

教育信息可度量数量信息表示与抽取模型

  • 构建可度量数量信息的语义表示模型，并建立可度量数量信息抽取的一般性技术标准，推动可度量数量信息的机器处理发展。
  • 主要内容： -可度量数量信息国际标准研制 -可度量数量信息的算法设计 -可度量数量信息的系统开发 -可度量数量信息标准的推广应用

广东省高考大数据分析挖掘与可视化

  • 广东省高考成绩的科学量化统计与分析，从不同维度了解高考基本情况，分析学生高考学习状况。
  • 主要内容： - 根据现代教育测量学的原理和方法，做好高考评价工作，对高考考试结果进行科学的量化统计与分析，可帮助基层从不同维度了解高考的基本情况，有效诊断学生的学习状况，找到教育教学中的优势和不足，为提高教学质量和促进学生全面发展提供依据，为教育行政部门教学决策提供参考

基于智能组卷的在线考试及分析平台

  • 构建领域题库，形成基于多种智能组卷算法的在线考试系统，服务随时随地的在线考试，并形成考试成绩与考试过程的大数据分析报告，形成可视化可交互结果展示。
  • 主要内容： - 领域题库构建系统 - 智能组卷算法（难易度、内容区间、知识点分布等） - 实时的、全过程的、分章节的考试管理 - 成绩统计与导出的定制化、个性化 - 考试大数据分析与可视化

儿童自动问答教育机器人

  • 构建面向3-7岁儿童的知识型问答机器人，服务儿童成长陪伴、知识学习、故事娱乐等。
  • 主要内容： - 人机交互机制 - 自动问答模型 - 知识图谱构建及应用 - 问题目标识别、意图分析 - 常问问答重用技术 - APP、手环、终端屏、机器人等多模式
